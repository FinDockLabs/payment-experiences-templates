# Experience Cloud Site Template
This folder contains an example Experience Cloud site that can serve as a starting point for building a donation page using Experience Cloud and FinDock. The site includes space for a donation screen flow as well as additional content. The included flow templates can be found in the [flow-templates](../flow-templates/README.md) folder and can be added to the site to create a donation flow. The site is meant to be customized and extended to fit the specific use case and requirements of your page.

## Installation
1. Create a new Experience Cloud site with the "Build your own (LWR)" template and the name "Donation Page".
2. Press the buttons to deploy the repo to the org.
3. Customize the page in the Experience Cloud builder
4. Publish the Experience Cloud
